Project Overview

This system uses deep learning to predict wildfire spread patterns based on multi-spectral satellite imagery from Landsat-8. The model analyzes 12 different environmental channels to generate pixel-level predictions of fire probability.

Model: Attention U-Net

The model is based on the U-Net segmentation architecture with attention mechanisms. This design excels at:

  • Capturing multi-scale spatial features from 12-channel input
  • Focusing on fire-critical regions with attention gates
  • Preserving fine-grained details with skip connections
  • Outputting pixel-level probability maps (0-1)
